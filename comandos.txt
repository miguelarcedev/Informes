# Comandos para hacer copia de respaldo de BD de Postgre. 

Directoria donde se encuanta las BD:
cd c:\Archivos de programa\PostgreSQL\16\bin

psql -U postgres     
\l

Para hacer Backup:
pg_dump -U <tu_usuario> -d <nombre_de_la_base_datos> -f <nombre_archivo_backup.sql>

Para restablecer el respaldo guardado:

psql -U nombre_usuario -d nombre_base_de_datos -f ruta/al/tu/archivo.sql
